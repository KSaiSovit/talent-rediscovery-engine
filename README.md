# Talent Rediscovery + Re-Engagement Engine

The Talent Rediscovery Engine is an AI-powered recruitment tool designed to unlock the hidden value in your existing Applicant Tracking System (ATS) or CSV databases. Instead of spending on new sourcing, this engine intelligently matches and ranks your "dormant" candidate pool against new Job Descriptions (JD).

## Setup

* pip install -r requirements.txt
* streamlit run app.py

## Usage

* Upload: Upload a CSV file (use sample_candidates.csv or your own data).
* Define: Paste a job description into the text area.
* Process: Click "Rank Candidates" to trigger the scoring engine.
* Review: Filter by priority, edit outreach messages, and export your shortlist.
* Reachout message: Recheck the messages and export the final file
